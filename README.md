## üìÅ Project: Adversarial Tool Research 
**Purpose:** To deconstruct malware techniques by building a controlled proof-of-concept keylogger with basic Command & Control (C2) functionality. This project explores:
- Keystroke capture mechanisms
- Data exfiltration methods
- Basic persistence techniques
- Defensive detection strategies

**Key Learning Outcomes:**
- Understanding how keyloggers operate at a code level
- Analyzing network covert channels
- Developing detection rules and threat-hunting queries
- Applying MITRE ATT&CK techniques (e.g., T1056.001, T1041)
